# fantasy-valorant-bot

## Setup

### Dependencies

Install dependencies.

```sh
pip install --upgrade setuptools
pip install -e .
```

May require sudo.

### Token

Create a file called `.env` in the root directory of this repository.

```sh
DISCORD_TOKEN="bot_token_here"
```

## Startup

From the project root, run:

```sh
python3 main.py
```
